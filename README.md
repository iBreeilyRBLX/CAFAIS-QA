# CAFAIS-QA
This is the public repo for the Cascadian Armed Forces Discord Bot.

This repo is used in order to track possible bugs and issues and to be transparent to the community about underlying bugs. 

## How to Report Issues

We welcome community feedback! If you encounter a bug or have a feature request, please follow these guidelines:

If you've found a bug in the Cascadian Armed Forces Discord Bot, please help us by submitting a detailed bug report:

1. **Check Existing Issues**: Before creating a new bug report, please search [existing issues](../../issues) to see if the problem has already been reported.

2. **Create a Bug Report**: If the issue hasn't been reported, [create a new bug report](../../issues/new?template=bug_report.md) using our bug report template.

3. **Provide Details**: Include as much information as possible:
   - Clear description of the bug
   - Steps to reproduce the issue
   - Expected vs. actual behavior
   - Screenshots (if applicable)
   - Bot version and environment details

2. **Submit a Feature Request**: If it's a new idea, [create a feature request](../../issues/new?template=feature_request.md) using our template.

3. **Describe Your Idea**: Help us understand your suggestion:
   - What problem does it solve?
   - How should it work?
   - How would it benefit the community?
